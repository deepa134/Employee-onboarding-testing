# Angular Frontend (converted)

This folder contains an Angular version of the React onboarding frontend.

How to run:

1. Install dependencies:

```powershell
cd angular-frontend; npm install
```

2. Start dev server (requires `@angular/cli`):

```powershell
npx ng serve --open
```

Notes:
- The generated app is a minimal manual scaffold that mirrors the React routes and logic.
- You may install a specific Angular CLI globally if desired: `npm i -g @angular/cli`.
